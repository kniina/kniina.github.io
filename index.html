Summary: Latitude vs. X

The purpose of this data analysis and visualization project was to examine how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API and used the CitiPy Python libary to assemble a dataset of over 500 cities across the world.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather versus city latitude. The objective was to build a series of scatter plots showcasing the following relationships:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
